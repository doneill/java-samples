# Esri Callout
Simple app using [Esri ArcGIS Runtime for Java SDK](https://developers.arcgis.com/java/) to show lat/lon of mouse click on a Map.

## Clean the project

`./gradlew clean`

## Build the app

`./gradlew assemble`

## Run the app
`./gradlew run`

## All together now

`./gradlew clean run --info`
